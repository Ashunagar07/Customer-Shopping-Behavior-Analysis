# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior using Python, SQL, and Power BI.

## Tools Used
- Python
- Pandas
- MySQL
- Power BI

## Key Analysis
- Customer Segmentation
- Purchase Frequency Analysis
- Discount Usage Analysis
- Product Performance Analysis
- Sales Trends
- Subscription Behavior Analysis

## Dashboard
Interactive Power BI dashboard created to visualize customer insights, sales patterns, and shopping trends.

## Files Included
- Python Analysis Notebook
- SQL Queries
- Power BI Dashboard
- Dataset
